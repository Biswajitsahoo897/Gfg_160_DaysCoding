# Reverse a Linked List

## Problem Statement
Given the head of a linked list, the task is to reverse this list and return the reversed head.

---

## Examples

### Example 1:
**Input:**  
head: `1 -> 2 -> 3 -> 4 -> NULL`  
**Output:**  
head: `4 -> 3 -> 2 -> 1 -> NULL`  
**Explanation:**  
The linked list is reversed.

### Example 3:
**Input:**

`2 -> NULL`
